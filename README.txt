MusYamat
Es un software creado para almacenar datos de un museo de guerra.
Fue hecho para el Taller de Programación del profesor German Andrés Riveros Sepulveda.

Para empezar
Tan solo debe tener el instalador de MusYamat listo.

Prerequisitos
Se necesita el tener Java instalado en su computadora para poder usar este software.

Instalación
1.-Después de abrir el instalador de MusYamat debe elegir el idioma del instalador.

2.-Debe aceptar los términos y condiciones de MusYamat antes de continuar con la instalación.

3.-Debe elegir la carpeta en la que se instalará MusYamat.

4.-Puede elegir si desea crear un acceso en el escritorio y si desea abrir MusYamat apenas termine la instalación.

5.-Esperar a que se instale MusYamat y salir del instalador.
